# Cputroller

GNU/Linux 下查看调节 CPU 的策略、风扇转速的图形化工具。

能调节 CPU 的策略为性能、按需、节能模式，能手动调节风扇转速或开启自动调节。

**说明**

本质上只是 cpufrequtils 和内核文件的前端，没有技术含量。

但是命令行的操作对绝大部分普通用户来说不友好，所以把它做成了可视化应用。

**开发环境**

deepin V20

**安装方法**

首先需要安装依赖，在终端中运行`sudo apt install -y cpufrequtils`。
